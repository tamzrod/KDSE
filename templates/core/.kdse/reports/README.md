# Reports Directory

This directory contains generated reports and analysis artifacts.

## Purpose

The reports directory stores:
- Audit reports
- Analysis reports
- Post-mortems
- Status reports

## Report Types

### Audit Reports

Generated through `kdse audit`

### Analysis Reports

Generated through evidence analysis

### Post-Mortems

Created after incidents or significant events

### Status Reports

Generated periodically for stakeholders

## Usage

Reports should be:
- Based on evidence from the workspace
- Linked to source artifacts
- Dated and versioned
- Actionable with clear recommendations

## Guidelines

- Include evidence citations
- Track report versions
- Archive old reports
- Share with stakeholders
